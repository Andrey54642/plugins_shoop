# WordPress Shop Map

MVP-плагин карты магазинов на WordPress.

## Точка входа

- `wordpress-shop-map.php` - основной файл плагина.

## Структура данных

- Магазины: CPT `sm_store`
- Бренды: 3 фиксированных слота в опциях плагина
- Город, адрес, бренды, `external_id`, координаты и override URL хранятся в post meta

## Админка

- `Магазины -> Настройки` - API-ключ Яндекс.Карт и настройки 3 фиксированных брендов
- `Магазины -> Импорт Excel` - загрузка `.xls` / `.xlsx` и импорт магазинов
- `Магазины -> Шорткоды` - справка по доступным шорткодам и приоритету URL кнопки

## Установка

1. Скопируйте папку `wordpress-shop-map` в `wp-content/plugins/`.
2. Активируйте плагин в админке WordPress.
3. Откройте `Магазины -> Настройки` и задайте:
    - API-ключ Яндекс.Карт
    - названия, логотипы и default URL 3 фиксированных брендов
    - default URL кнопки для каждого бренда
4. При необходимости используйте `Магазины -> Импорт Excel` для массовой загрузки магазинов.

## Использование

- Добавьте магазины в разделе `Магазины`.
- Для магазина укажите:
  - название
  - город
  - адрес
  - бренды
  - координаты или дождитесь автогеокодирования
- При массовой загрузке через Excel используйте колонки `title`, `city`, `address`, `brands`, а для надежного обновления записей добавляйте `external_id`.
- Вставьте шорткод на страницу:

```text
[stores_map]
```

### Override URL кнопки

```text
[stores_map button_url="https://example.com/buy"]
```

При override текст кнопки меняется на `Купить`.

## Импорт из Excel

Импортатор поддерживает `.xls` и `.xlsx`. Первый непустой лист используется автоматически.

Обязательные колонки:

- `title` - название магазина
- `city` - город
- `address` - адрес
- `brands` - один или несколько брендов

Необязательные колонки:

- `lat` - широта
- `lng` - долгота
- `external_id` - внешний идентификатор для однозначного обновления записи

Допустимые заголовки для колонок описаны на странице `Магазины -> Импорт Excel`.

Пример строки:

```text
title | city | address | brands | external_id
Магазин на Тверской | Москва | Тверская улица, 1 | brand1, brand2 | msk-001
```
